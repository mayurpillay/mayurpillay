Zeppln homepage

Zeppln was my idea for a site that would allow people to share things that they made. It would build on the community aspect of sharing content. Almost like a Dribbble for DIY type stuff. The design focused on creating an image rich experience. I felt that images were the most important assets on the page and would create the most engagement before a click-through. I chose a grid layout because I also wanted to create an environment that fostered discovery and inspiration. I added the "Shuffle" link to the header thinking that clicking it would reorganize the page randomly, I felt that this also contributed to the discovery component. The name Zeppln came to mind because Zeppelins were known for their intricate and advanced design and construction allowing them to be longer and carry heavier loads.

Zeppln Product Page

Each project uploaded by a user would have a product page. The goal of the product page was to allow users to see a project's stats, description, steps / materials, cost and comment on it. The "? ask" button was meant to add a marketplace component to the site where users can ask other users to build something for them. It would have been interesting to see how this functionality would be used. I could see it either as a status symbol (the more asks someone has the better) or as an actual tool that people can use to get things built by other people. The layout of the page focused on the project image. I chose to elevate the main stats with a drop shadow because I envisioned them to qualify how a project is perceived by the community.
